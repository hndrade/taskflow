⚙️ Configurar TaskFlow

Para sincronizar suas tarefas entre PC e celular, conecte ao Supabase (gratuito e permanente). Ou use o modo local.

1. Acesse supabase.com → crie um projeto grátis
2. Vá em Project Settings → API e copie a Project URL e a anon public key
3. No Table Editor, crie uma tabela chamada tasks com as colunas: id (text, PK), user_id (uuid), data (jsonb). Ative RLS e crie policies para SELECT/INSERT/UPDATE/DELETE com auth.uid() = user_id
4. Em Authentication → Providers, ative Google ou Email Magic Link
5. Cole as credenciais abaixo e clique em Conectar
Os dados ficam salvos no localStorage do navegador deste dispositivo.
Ideal para uso em um único dispositivo. Não sincroniza entre PC e celular.
